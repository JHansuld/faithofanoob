This is my guide on how to install Jekyll to use with Github Pages.

I am going to go over how to do this in linux command line with links to the download files as well. All info can be found at jekyllrb.com. First we have to install some prerequsites.

Install Ruby

  1. Download Ruby here or in linux command line:

    $ sudo apt-get install ruby-full build-essential zlib1g-dev

  2. Verify you have the latest version with:

    $ ruby -v

RubyGems

If you installed Ruby with the command line instructions above, you may skip this step.

  1. Download RubyGems here

Linux install

  1. Create a folder to install your Gem files into. Never install it as a root user. Use:

    $ mkdir gems

  2. add your variables to the bashrc file by typing these commands. NOTE: $ indicates a new line so press enter for the previous command

    $ echo '# Install Ruby Gems to ~/gems' >> ~/.bashrc

    $ echo 'export GEM_HOME="$HOME/gems"' >> ~/.bashrc

    $ echo 'export PATH="$HOME/gems/bin:$PATH"' >> ~/.bashrc

    $ source ~/.bashrc/code

  3. The last step is to install Jekyll:

    $ gem install jekyll bundler

Windows install

  1. Press the windows key and type:

    Command prompt

  2. Select the Command Propt
  3. In Command Prompt type:

    bash

  4. Now we need to update our repository lists and packages. Type:

    $ sudo apt-get update -y && sudo apt-get upgrade -y

  5. Next we make sure RubyGems is up to date with:

    $ gem update

  6. Finally we install Jekyll:

    $ gem install jekyll bundler
